Admin Tools

This page centralizes operational tasks, repair jobs, diagnostics, and maintenance actions. Routine controls are separated from broad-impact actions so the feed page stays focused on reviewing stories.

WARNING!

Tools marked with ⚠ Danger can rewrite large amounts of stored data, change grouping or summaries site-wide, or make troubleshooting harder if run casually. Use those only when you have a concrete repair goal.

Tools marked with ⚠ Broad change are less destructive, but they still update many existing records at once and can change how content is categorized or interpreted.

Operational
Short-running actions you might use during normal day-to-day monitoring or when the AI side is offline or behind.
Wake Ollama

Sends the Wake-on-LAN packet to the Ollama machine so AI tasks can resume if the box is asleep.

Safe
Run AI Catch-up

Fills in missing AI outputs for existing stories and articles without running a full scheduled pipeline.

Safe
Scrape Missing

Attempts to fetch readable article content for recent articles that are missing content or still marked failed/pending.

Safe
Reindex Search

Rebuilds the Meilisearch story and article indexes from the database so admin search stays fast and can search both stories and articles.

Maintenance
Idle
Search index has not been rebuilt in this app session yet.
Content Repair
Use these when existing stories need repair or normalization after a ranking, grouping, topic, or outlet-bias issue.
Reclassify Topics

Re-runs topic classification so stored stories/articles can be reassigned to better topics after prompt or model changes.

⚠ Broad change
Rebuild Story Grouping

Forces articles back through grouping logic. Use this only when story clustering is clearly wrong and you want to rebuild it.

⚠ Danger
Rebuild Story Summaries

Clears and regenerates summaries/deep reports across stories. Useful after major prompt or scraper improvements.

⚠ Danger
Sync AllSides Ratings

Refreshes outlet bias from the built-in AllSides mapping and propagates updated outlet scores to related articles.

Maintenance
Scraper Diagnostics
Navigate to live scraper telemetry or run targeted auditing when you suspect blocklist or content-quality problems.
Scrape Blocklist

Opens the dedicated blocklist page with recent domain trouble, retry cache information, and unblock controls.

Open Scrape Blocklist
Audit Stored Scrapes

Re-audits existing stored article content to catch bad login pages, duplicate junk, or weak content that should be repaired.

⚠ Broad change
Data Maintenance
These actions change canonical stored data and should be used carefully, ideally when you already know the inconsistency you are fixing.
Merge Duplicate Outlets

Renames outlets to their normalized forms and merges duplicates, reassigning articles to a single canonical outlet.
