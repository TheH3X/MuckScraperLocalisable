Admin Tools

This page centralizes operational tasks, repair jobs, diagnostics, and maintenance actions. Routine controls are separated from broad-impact actions so the feed page stays focused on reviewing stories.

WARNING!

Tools marked with ⚠ Danger can rewrite large amounts of stored data, change grouping or summaries site-wide, or make troubleshooting harder if run casually. Use those only when you have a concrete repair goal.

Tools marked with ⚠ Broad change are less destructive, but they still update many existing records at once and can change how content is categorized or interpreted.
